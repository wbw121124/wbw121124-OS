# wbw121124 OS
## 1. 介绍
wbw121124 OS 是一个基于Windows API、免费且自由的操作系统，它具有以下特点：
- 使用C++语言编写
- 类似于cmd命令行

## 2. 环境配置
- 操作系统：Windows >= Windows 10
- 编译器：MinGW or GCC

## 3. 使用方法
- 下载源码
- 编译
	- 编译选项：`-std=c++14`
- 运行
